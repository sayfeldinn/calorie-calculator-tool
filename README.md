## ‚ú® Features
- **Accurate Calculations**: Uses the highly reliable Mifflin-St Jeor equation for basal metabolic rate (BMR).
- **Unit Support**: Seamlessly switch between Metric (kg/cm) and Imperial (lbs/in) inputs.
- **Dynamic Input Limits**: Prevents unrealistic entries and validates values dynamically.
- **Glassmorphic UI**: Beautiful, modern aesthetic with neon green accents embedded on a dark background.
- **Instant Plan Generation**: Get comprehensive insights into the total daily calories you need to consume to hit your target.
- **Responsive Design**: Carefully optimized for an excellent user experience on mobile, tablet, and desktop devices.
## üöÄ Getting Started
First, ensure you have Node.js installed on your machine.
Clone the underlying project repository, then navigate locally into the directory:
```bash
git clone https://github.com/sayfeldinn/calorie-calculator-tool.git
cd calorie-calculator-tool
```
Install the required dependencies:
```bash
npm install
```
Start the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
